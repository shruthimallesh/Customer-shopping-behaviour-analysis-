# Customer-shopping-behaviour-analysis-

Data analytics project showcasing customer behaviour analysis using python,sql and powerbi.


Overview
This project demonstrates an end-to-end data analytics workflow, covering data loading, exploratory data analysis (EDA), data cleaning, SQL querying, and business insights visualization through Power BI.
The goal of this project is to transform raw data into meaningful insights that support data-driven decision-making.
